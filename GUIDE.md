# 프라나 AI — 사용 설명서

## 개요

프라나 AI는 블루아카이브 게임 정보를 제공하는 웹사이트입니다. 정보글 작성/조회, 엘리그마 계산기, 제조 노드 기대값 계산기 등의 기능을 제공합니다.

---

## 1. 회원가입 및 로그인

### 회원가입

1. 사이드바 하단의 **로그인** 클릭
2. 로그인 페이지 하단의 **회원가입** 링크 클릭
3. **닉네임**, **이메일**, **비밀번호**(6자 이상) 입력 후 **회원가입** 버튼 클릭
4. 가입 완료 후 **승인 대기** 상태가 됩니다
5. 관리자가 승인하면 역할에 따라 활동이 가능합니다

### 로그인

1. 사이드바 하단의 **로그인** 클릭
2. 이메일, 비밀번호 입력 후 **로그인** 버튼 클릭
3. 로그인 후 사이드바 하단에 닉네임과 **로그아웃** 버튼이 표시됩니다

### 로그아웃

- 사이드바 하단의 **로그아웃** 버튼 클릭

---

## 2. 역할별 권한

| 역할 | 정보글 읽기 | 정보글 작성/수정/삭제 | 카테고리 관리 | 유저 관리 |
|------|-----------|---------------------|-------------|---------|
| 승인 대기 (pending) | O | X | X | X |
| 사용자 (user) | O | X | X | X |
| 부관리자 (editor) | O | O (관리자 글 제외) | X | X |
| 관리자 (admin) | O | O (모든 글) | O | O |

- 부관리자는 관리자가 작성한 글을 수정/삭제할 수 없습니다
- 관리자는 모든 글을 수정/삭제할 수 있습니다

---

## 3. 정보글

### 정보글 목록 보기

1. 사이드바에서 **정보글** 클릭
2. 상단 카테고리 탭으로 필터링 가능 (**전체**, **공지**, 기타 카테고리)
3. 공지 카테고리의 글은 항상 **최상단에 빨간 배경**으로 표시됩니다
4. 글 행을 클릭하면 상세 페이지로 이동합니다

### 정보글 작성 (부관리자/관리자)

1. 정보글 목록 페이지에서 **글 작성** 버튼 클릭
2. **제목** 입력
3. **카테고리** 선택
4. **본문** 작성 — 리치 텍스트 에디터(WYSIWYG)를 사용합니다
   - 텍스트 서식: 굵게(B), 기울임(I), 밑줄(U), 취소선(S)
   - 제목: H1, H2, H3
   - 글자 색상: 색상 선택기로 변경, "기본색" 버튼으로 초기화
   - 정렬: 좌/중/우
   - 목록: 순서 없는 목록(●), 순서 있는 목록(1.)
   - 삽입: 링크, 이미지(URL), YouTube 동영상(URL), 인용, 코드블록, 구분선
   - 실행취소/다시실행: ↩ / ↪
5. **작성하기** 버튼 클릭

### 정보글 수정 (부관리자/관리자)

1. 정보글 목록에서 **수정** 버튼 클릭, 또는 상세 페이지에서 **수정** 버튼 클릭
2. 내용 수정 후 **수정하기** 버튼 클릭

### 정보글 삭제 (부관리자/관리자)

1. 정보글 목록에서 **삭제** 버튼 클릭, 또는 상세 페이지에서 **삭제** 버튼 클릭
2. 확인 팝업에서 **확인** 클릭
3. 삭제된 글은 완전히 제거되지 않고 비활성화 처리됩니다 (관리자가 복원 가능)

## 4. 관리자 전용 기능

### 카테고리 관리

1. 정보글 목록 페이지에서 **카테고리 관리** 버튼 클릭
2. 새 카테고리 이름 입력 후 **추가** 버튼 클릭
3. 기존 카테고리의 **삭제** 버튼으로 삭제 가능
4. **전체**와 **공지** 카테고리는 기본 카테고리로 삭제할 수 없습니다

### 유저 관리

1. 사이드바의 **유저 관리** 클릭
2. **승인 대기** 섹션: 신규 가입자 목록이 표시됩니다
3. 역할 변경 드롭다운에서 원하는 역할 선택:
   - **승인 대기** → 가입 후 기본 상태
   - **사용자** → 정보글 읽기만 가능
   - **부관리자** → 정보글 작성/수정/삭제 가능
   - **관리자** → 모든 기능 사용 가능
4. **비활성화** 버튼으로 유저를 비활성화할 수 있습니다
5. 비활성화된 유저는 하단 섹션에서 **복원** 버튼으로 복원 가능합니다

### 삭제된 글 관리

1. 정보글 목록 페이지에서 **삭제된 글** 버튼 클릭
2. 삭제된 글 목록이 표시됩니다 (취소선으로 구분)
3. **복원** 버튼으로 삭제된 글을 다시 활성화할 수 있습니다

### 수정 이력 (로그) 조회

1. 정보글 목록에서 **로그** 버튼 클릭 (보라색)
2. 해당 글의 전체 이력을 확인할 수 있습니다:
   - **생성**: 글이 처음 작성된 시점
   - **수정**: 글이 수정된 시점
   - **삭제**: 글이 삭제된 시점
   - **복원**: 글이 복원된 시점
3. 각 이력에 **작업자 닉네임**과 **작업 일시**(초 단위까지)가 기록됩니다

## 5. 계산기

### 엘리그마 계산기

1. 사이드바에서 **엘리그마 계산기** 클릭
2. **현재 성급**과 **목표 성급** 선택
3. **현재 보유 엘레프(조각)** 수 입력
4. **엘레프 1개당 가격**과 **잔여 구매 횟수** 설정
5. **계산하기** 버튼 클릭
6. 결과: 상점에서 구매할 조각 수 + 필요한 총 엘리그마가 표시됩니다

### 제조 노드 기대값 계산기

1. 사이드바에서 **제조 계산기** 클릭

**모드 1: 노드 → 아이템 기대값**
1. 1차/2차/3차 노드를 각각 드롭다운에서 선택
2. 선택한 노드 조합의 아이템별 기대 확률이 자동 표시됩니다
3. 아이템 검색창으로 특정 아이템을 필터링할 수 있습니다
4. 정렬: 기대 확률순 / 티어 오름차순 / 티어 내림차순

**모드 2: 아이템 → 최적 노드 추천**
1. 상단의 **아이템 → 최적 노드 추천** 탭 클릭
2. 원하는 아이템을 검색하고 선택
3. 해당 아이템을 얻기 위한 최적의 노드 조합이 추천됩니다

## 6. 모바일 사용

- 모바일에서는 상단 헤더의 **햄버거 메뉴(≡)** 버튼을 클릭하면 사이드바가 열립니다
- 메뉴 항목을 선택하거나 배경을 클릭하면 사이드바가 닫힙니다
